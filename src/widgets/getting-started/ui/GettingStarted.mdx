# Установка и использование

Эта страница описывает, как подключить `ui-kit` в проект-потребитель и какие шаги нужны для корректной работы стилей, компонентов и опциональных addons.

`src/widgets` используется только для страниц и виджетов Storybook. Код из этой папки не входит в публичное API библиотеки, поэтому в приложениях нужно импортировать пакет через опубликованные entrypoint'ы.

## Установка

Установите библиотеку:

```bash
pnpm add ui-kit
```

Установите обязательные peer dependencies:

```bash
pnpm add sonner @tabler/icons-react
```

Если используете `Form` или примеры с валидацией, установите опциональные зависимости:

```bash
pnpm add @hookform/resolvers react-hook-form valibot
```

```css
@custom-variant dark (&:where(.dark, .dark *));

@theme inline {
  --font-sans: var(--font-sans);

  --color-border: var(--border);
  --color-ring: var(--ring);

  --color-background: var(--background);

  --color-card: var(--card);
  --color-card-foreground: var(--card-foreground);

  --color-primary: var(--primary);

  --color-constructive-foreground: var(--constructive-foreground);
  --color-informative: var(--informative);
  --color-informative-foreground: var(--informative-foreground);
  --color-warning: var(--warning);
  --color-warning-foreground: var(--warning-foreground);
  --color-destructive: var(--destructive);
  --color-destructive-foreground: var(--destructive-foreground);
  --color-accent: var(--accent);

  --color-destructive-active: var(--destructive-active);
  --color-destructive-foreground-active: var(--destructive-foreground-active);
  --color-destructive-disabled: var(--destructive-disabled);
  --color-destructive-foreground-disabled: var(--destructive-foreground-disabled);

  --color-accent-foreground-disabled: var(--accent-foreground-disabled);

@layer base {
  * {
    @apply scrollbar-thin scrollbar-thumb-[#dedfe1] scrollbar-track-background border-border outline-hidden dark:scrollbar-thumb-[#3a3d45];
  }

  :focus-visible {
    @apply ring-3 ring-ring;
  }

  body {
    @apply bg-background text-foreground antialiased;
  }
}
```

`ui-kit/styles` уже содержит Tailwind CSS 4 `@source` для собранных модулей библиотеки. Не добавляйте отдельный `@source` на `node_modules/ui-kit`.

Библиотека использует корпоративный шрифт `pnfont`. Файлы шрифтов не входят в пакет, поэтому их нужно разместить в проекте-потребителе:

```txt
public/fonts/pnfont/
```

## Импорты

Конфигурация импортируется из `ui-kit/config`:

```ts
import { FIELD_LENGTHS, MEDIA_QUERIES, OTP_PATTERNS } from 'ui-kit/config'
```

Утилиты, типы и хуки импортируются из `ui-kit/lib`:

```ts
import {
  classNames,
  formatCreditCard,
  getRequiredEnv,
  getFormDigitsValue,
  isValidCreditCardNumber,
  parseCreditCardExpiry,
  useMeasure,
  useMediaQuery,
  type SelectOption,
} from 'ui-kit/lib'
```

Addon `Form` импортируется из отдельного subpath:

```ts
import { Form } from 'ui-kit/addons/form'
```

Core-компоненты импортируются из `ui-kit/core`:

```ts
import { BUTTON_SIZES, BUTTON_VARIANTS, buttonVariants, type ButtonProps, type ButtonSize, type ButtonVariant } from 'ui-kit/core'
```

## Локальная разработка

Установите зависимости репозитория:

```bash
pnpm install
```

Перед завершением изменений используйте узкие проверки:

```bash
pnpm lint
pnpm check
```

Запустите Storybook:

```bash
pnpm storybook
```

Если менялись Storybook-страницы, документация или визуальные примеры, проверьте сборку Storybook:

```bash
pnpm build-storybook
```
